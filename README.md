# FrankenPHP Docker Compose

## Features:
- `Dockerize` - {`sqlite`, `mysql`, `mariadb`, `postgres`, `sqlsrv`, `mongodb`}
- `PHPMyAdmin`
- `PGAdmin`
- `Mongo Express`
- `Env`
- `GitHub Actions`
- `Gitlab CI`
- `Bitbucket Pipelines`

## Supported Databases
- `sqlite`
- `mysql`
- `mariadb`
- `postgres`
- `sqlsrv`
- `mongodb`

## Install && Start
```
docker compose -f compose.yaml up -d
```
